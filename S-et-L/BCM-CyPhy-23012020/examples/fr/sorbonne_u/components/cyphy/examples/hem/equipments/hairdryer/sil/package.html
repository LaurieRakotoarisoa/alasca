This package defines DEVS SIL simulation models used in the hair dryer component.

This simulation model uses two complementary atomic models:

  1. The HairDryerSILModel keeps track of the operating mode of the hair dryer by peeking the component for its current operating and associates to the operating mode the level of electricity consumption. It exports an event giving its current consumption emitted each time a change is observed in this level.
  2. The HairDryerSILUserModel implements usage scenarios calling the hair dryer services to change its operating mode.

The coupled model HairDryerSILCoupledModel connects the two previous atomic models. It can be executed as a stand alone SIL simulation to perform unit testing of the component.

To run a SIL simulation after the implementation of the component, use the class CVM in the package unittest.
